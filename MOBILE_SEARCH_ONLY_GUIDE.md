# 모바일 검색 전용 모드 가이드

## 개요
모바일에서는 매물 등록 없이 검색 기능만 사용할 수 있도록 최적화되었습니다.

## 모바일 검색 전용 모드 특징

### ✅ 사용 가능한 기능
- **매물 검색**: 제목, 주소, 매물번호로 검색
- **필터링**: 거래유형, 매물종류, 면적, 가격, 보증금 필터
- **지도 조회**: 매물 위치 확인 및 지도 탐색
- **매물 상세 조회**: 매물 정보 및 이미지 확인
- **매물번호 검색**: 특정 매물번호로 직접 검색

### ❌ 사용 불가능한 기능
- **매물 등록**: 새 매물 추가 불가
- **매물 수정**: 기존 매물 정보 수정 불가
- **매물 삭제**: 매물 삭제 불가
- **관리자 기능**: 모든 관리자 기능 비활성화

## 모바일 감지 기준

### 화면 크기 기준
- **768px 이하**: 모바일로 인식
- **769px 이상**: PC로 인식

### 자동 모드 전환
- 화면 크기가 변경되면 자동으로 모드 전환
- 모바일에서 PC로 전환 시 모든 기능 활성화
- PC에서 모바일로 전환 시 검색 전용 모드로 전환

## 사용 방법

### 1. 기본 검색
1. 상단 검색창에 키워드 입력
2. 엔터키 또는 검색 버튼 클릭
3. 검색 결과가 지도와 목록에 표시

### 2. 고급 필터링
1. 필터 버튼 클릭
2. 원하는 조건 선택:
   - 거래유형: 매매/임대
   - 매물종류: 상가/사무실/건물/기타
   - 면적: 평수 범위 선택
   - 가격: 매매가 또는 임대료 범위
   - 보증금: 보증금 범위
3. 필터 적용 후 결과 확인

### 3. 매물번호 검색
1. 매물번호 입력창에 정확한 매물번호 입력
2. 해당 매물이 자동으로 선택되어 표시
3. 지도에서 해당 위치로 자동 이동

### 4. 지도 탐색
- **줌**: 핀치 제스처 또는 줌 버튼 사용
- **이동**: 드래그로 지도 이동
- **마커 클릭**: 매물 정보 확인
- **클러스터 클릭**: 여러 매물 목록 확인

## 모바일 최적화 기능

### 터치 친화적 UI
- 큰 버튼과 터치 영역
- 스와이프 제스처 지원
- 핀치 줌 지원

### 반응형 레이아웃
- 화면 크기에 따른 자동 레이아웃 조정
- 세로/가로 모드 지원
- 다양한 화면 해상도 대응

### 성능 최적화
- 이미지 지연 로딩
- 무한 스크롤 지원
- 캐시 최적화

## 검색 팁

### 효과적인 검색어
- **지역명**: "강남구", "마포구", "홍대"
- **상권명**: "홍대입구", "신촌", "명동"
- **건물명**: "롯데월드타워", "63빌딩"

### 필터 조합 활용
- 거래유형 + 매물종류 조합
- 면적 + 가격 범위 조합
- 보증금 + 임대료 조합

### 매물번호 활용
- PC에서 등록된 매물의 정확한 번호 사용
- 매물번호는 대소문자 구분 없음
- 부분 검색 가능

## 문제 해결

### 검색 결과가 없는 경우
1. 검색어 철자 확인
2. 필터 조건 완화
3. 지도 범위 확대

### 이미지가 표시되지 않는 경우
1. 네트워크 연결 확인
2. 페이지 새로고침
3. 브라우저 캐시 삭제

### 지도가 로드되지 않는 경우
1. 인터넷 연결 확인
2. Google Maps API 키 확인
3. 브라우저 권한 설정 확인
